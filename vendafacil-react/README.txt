COMO ABRIR O PROJETO:

1. Extraia esta pasta.
2. Abra a pasta no VS Code.
3. Abra o terminal dentro da pasta.
4. Digite:
   npm install
5. Depois digite:
   npm run dev
6. Abra o link que aparecer, normalmente:
   http://localhost:5173
